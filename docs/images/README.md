## Image Guidelines

- **Format:** PNG
- **Max size:** 500KB per image
- **Resolution:** 1920x1080 or lower
- **Theme:** Use consistent Home Assistant theming
- **Focus:** Crop to show relevant UI elements clearly

## How to Take Screenshots

1. Open Home Assistant in your browser
2. Navigate to the relevant screen
3. Use browser screenshot tool or OS screenshot
4. Crop to focus on the relevant UI
5. Optimize file size (use tools like `pngquant` or online optimizers)
6. Save with the filename from the checklist above

## Usage in Documentation

Reference images in markdown like this:

```markdown
![LLM Settings](images/config-llm-settings.png)
```
